# CodeAlpha Event Registration System

## Description
A simple Event Registration System built using Node.js, Express.js and MongoDB.

## Features
- Add Event
- View All Events
- Register for an Event
- View All Registrations
- Cancel Registration

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

```bash
npm install
npm start
```

## API Endpoints

### Events

POST /events/add

GET /events

GET /events/:id

### Registrations

POST /registrations

GET /registrations

DELETE /registrations/:id
